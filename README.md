# HTLM Video Parser
A simple collection of scripts based on YTDL bins for testing the parsing of and reading video streams for eductaional use of stream ingestion and translation with FFMPEG.

## Installation
Requires python, ffmpeg, yt-dlp, and "lxml" python library

## Usage
To use is quite simple, once setup:
```bash
./ytdl.sh "LINK TO WEBPAGE TO PARSE"
```
